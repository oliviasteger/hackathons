Vocabulary Builder

Learn new vocabulary using Merriam Webster. This game is created by teens, for teens in order to expand their vocabulary!

You will be given the definition of the word, and based off of the definition, you will have to find the word. If you guess the correct word, you can play a game. If you guess incorrectly, you have the chance to guess again.
